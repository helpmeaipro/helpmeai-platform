# HelpMeAI AI Router

## Purpose

The AI Router is responsible for selecting the correct AI department for every user request.

## Responsibilities

- Understand user intent
- Classify the request
- Select the correct department
- Forward the task
- Collect the result
- Send the response back to the CEO AI
