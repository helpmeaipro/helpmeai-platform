# HelpMeAI Documentation Standard

## Rules

- Every folder must contain a README.md
- Every module must have its own documentation.
- Every AI Engine must have architecture documentation.
- Every API must be documented.
- Every database table must be documented.
- Every workflow must have a flow diagram.
- Every major decision must be recorded.
